实验题目：利用云平台搭建个人博客
实验学时：2 实验日期：2025/3/12

实验目的：本实验旨在通过实践操作，掌握基于云计算环境的网站部署技能。

具体包括：构建环境并实现个人博客/网站的搭建，撰写实验报告，并将这份报告作为一篇博文发布在自己的博客 / 网站上。

硬件环境： 

联网的计算机一台

软件环境：

Windows or Linux

实验步骤与内容：

1. 安装Git和Node.js

 由于我之前已经安装过，不再重新安装。

 

 

2. 全局安装 hexo

 通过命令安装：npm i hexo-cli -g

 

3. 初始化一个项目

 命令：hexo init blog。系统中会生成项目文件夹blog

 

4. 运行

 通过命令运行：hexo server。博客初始页面如下：

 

 5.更换主题

 选择自己喜欢的主题，并通过clone命令安装主题

 我选择的主题为Materialized。打开blog下的_config.yml，将theme修改为自己主题的名称，具体如下：

 

 

 6.部署到github

 6.1生成个人访问令牌：

 登录到 GitHub。

 进入 Settings > Developer settings > Personal access tokens。

 点击 Generate new token，选择适当的权限并生成令牌。

 

 

 6.2编辑 _config.yml 文件，确保部署配置中使用了个人访问令牌

 

 

 6.3安装推送工具

 命令:npm install hexo-deployer-git –save

 

 6.4执行 Hexo 部署命令

 hexo clean && hexo g && hexo d

 可以在github上看到项目已经放到了仓库里

 

7. 写文章

 首先新建一个markdown文件：hexo new blog。接着使用markdown格式书写新建的blog.md。

 

 

结论分析与体会：

通过本次实验，我学会了如何搭建个人博客网站，在这个过程中，我学习了更多git的使用，也认识到了hexo这个博客框架。通过这次实践操作，我掌握基于云计算环境的网站部署技能。
